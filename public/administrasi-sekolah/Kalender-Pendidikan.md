# Kalender Pendidikan (Contoh Ringkas)

Berikut adalah contoh rincian kegiatan pada kalender pendidikan untuk semester ganjil (Tahun Ajaran Contoh).

| Bulan | Tanggal | Uraian Kegiatan |
|---|---|---|
| **Juli** | 15 - 17 Juli | Masa Pengenalan Lingkungan Sekolah (MPLS) untuk peserta didik baru. |
| | 18 Juli | Hari pertama KBM (Kegiatan Belajar Mengajar) efektif. |
| **Agustus** | 17 Agustus | Upacara HUT Kemerdekaan Republik Indonesia. |
| | 18 - 19 Agustus | Lomba kebersihan kelas dan kreativitas siswa memperingati 17-an. |
| **September**| 16 - 21 September| Penilaian Tengah Semester (PTS) / Asesmen Formatif Bersama. |
| **Oktober** | 28 Oktober | Upacara Hari Sumpah Pemuda. |
| | 31 Oktober | Rapat evaluasi bulanan dewan guru. |
| **November** | 25 November | Peringatan Hari Guru Nasional (Upacara dan Pentas Seni Siswa). |
| **Desember** | 2 - 7 Desember | Penilaian Akhir Semester (PAS) / Asesmen Sumatif Akhir Semester. |
| | 9 - 14 Desember | Pengolahan nilai e-Rapor, remedial, dan pengayaan. |
| | 20 Desember | Pembagian Rapor Semester Ganjil. |
| | 22 Des - 4 Jan | Libur Semester Ganjil. |

*Catatan: Jadwal ini adalah simulasi dan dapat disesuaikan dengan kalender pendidikan resmi dari Dinas Pendidikan setempat.*
